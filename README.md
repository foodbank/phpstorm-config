How to install
---

1. **File** -> **Import Settings**
2. Clone color schemes on your desktop [link to daylerees](https://github.com/daylerees/colour-schemes)
3. cd colour-schemes
4. mv jetbrains ~/Library/Preferences/PhpStorm2016.1/colors
